

#### 父组件向子组件传递数据

父组件 `Father.vue` 代码：

 ```html
 <template>
     <div class="box" style="padding-top:1rem;">
         <Child :data-content="mdata"></Child>
     </div>	
 </template>
 <script setup lang="ts">
     import Child from "Child.vue"
    	const mdata = [1,2,3]	//测试数据
 </script>
 ```

子组件`Child.vue`代码：

```html
<template>
    <div class="box" style="padding-top:1rem;">
        {{ data.data-content }}
    </div>	
</template>
<script setup lang="ts">
    //关键代码：通过defineProps提取数据
   	const data = defineProps(['data-content'])	//指定传递过来的数据名称props
</script>
```

结果：

```sh
{ "data-content": [ 1, 2, 3 ] }
```

##### 透传报错解决

```sh
#报错示例信息如下：
[Vue warn]: Extraneous non-props attributes (contentData, parentMethod) were passed to component but could not be automatically inherited because component renders fragment or text root nodes. 

#说明：
父组件向某个子组件传递属性，但是子组件并没有用defineProps接收此时就会触发透传Attributes。
而此时如果子组件如果有多个根节点，就会导致透传无法定位，控制台就会出现该警告错误。

#解决方法
1.子组件使用defineProps接收父组件传递的属性，或者父组件删除该属性
2.子组件<template></template>里最外层套一个<div>...</div>，确保只有一个根节点
```





#### 子组件向父组件传递

Father.vue

```html
<template>
    <div class="box" style="padding-top:1rem;">
        <Child ref="childRef"></Child>
    </div>	
</template>
<script setup lang="ts">
    import Child from "Child.vue"
   	const childRef = ref()
    console.log(childRef.value.a)		//获取属性
    console.log(childRef.value.test())	//获取方法
</script>
```

Child.vue

```html
<template>
    <div class="box" style="padding-top:1rem;">
        {{ data }}
    </div>	
</template>
<script setup lang="ts">
   	const a = 1
    const test()=>({
        return "test()..."
    })
    //关键代码
    defineExpose({a,test})		//暴露属性和方法给父组件
</script>
```
