#### 系统扫描

```
探查主机的操作系统类型
 --原理：根据返回的数据包中的不同的特征码判断
 nmap -O
开放端口监听的网络服务
 nmap -sV
```

#### nmap命令

```sh
nmap -sS -Pn x.x.x.x	#扫描ip，系统端口服务信息
nmap -sV -Pn x.x.x.x	#扫描版本信息
nmap -A x.x.x.x			#综合扫描（数据多，耗时长）
```
