#### 记一次vpn故障

> win11系统



**手贱操作：在cmd下使用命令`route delete *`清空了路由表** 

导致vpn无法连接，无法fq。但是可以上网

ping 本机ip正常

ping网关正常

ping百度正常

<font color=red>ping 127.0.0.1 失败 </font> 



**解决方法：** 

重置tcp/ip，cmd输入命令：

```sh
netsh int ip reset	#重置tcp/ip协议
```

按提示<font color=red>重启</font> 

重启后，能ping通127.0.0.1

vpn恢复正常
